# Wi-Fi Security

## Secured vs Unsecured Networks
- **Secured networks** use encryption to protect data (WPA/WPA2/WPA3).
- **Unsecured networks** are open and vulnerable to eavesdropping.

## Wi-Fi Protected Access (WPA)
- WPA/WPA2/WPA3 encrypts all traffic between devices and routers.
- Always use WPA2 or WPA3 where possible.

## Demo
- On your computer or phone:
  - Check Wi-Fi properties → see security type.
  - Avoid open Wi-Fi for sensitive tasks (banking, passwords).
